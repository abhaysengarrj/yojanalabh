# Release Notes

## Version 1.0.0 (Initial Release)

**What's New:**
- Launch of योजनालाभ — government scheme eligibility checker
- 21 real government schemes with eligibility rules (18 central + 3 new 2026 schemes)
- Hindi-first UI with simple 5-question form
- Eligibility matching engine with percentage scoring
- Works offline after initial data load
- Auto-update from GitHub (weekly checks)
- AdMob integration ready
- Material 3 design

**Schemes included:**
- PM-Kisan, PM Awas Yojana, PM Jan Dhan, NFSA, PM Fasal Bima
- PM Shram Yogi Maandhan, Sukanya Samriddhi, Janani Suraksha
- PM Matru Vandana, Atal Pension, PM Ujjwala, DAY-NULM
- MGNREGA, PM SVANidhi, DAY-NRLM, Vidya Lakshmi
- NSAP, PM Jan Aushadhi
- **NEW:** Lakhpati Didi Yojana, PM Vishwakarma Yojana, PM Surya Ghar Yojana

**Bug Fixes:**
- Fixed crash on tab switch (added mounted check)
- Fixed crash on devices without Google Play Services
- Fixed crash on slow network during link opening
- Set minSdk to 21 for 95%+ device coverage
